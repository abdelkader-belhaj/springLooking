1.Ouvrir PowerShell dans le dossier du projet

Ouvrir PowerShell puis aller vers le dossier du projet Spring Boot :

cd C:\place\projet

2.Définir les variables d’environnement (SMTP Gmail)

Configurer l’email et le mot de passe d’application Gmail :

$env:MAIL_USERNAME="bjgadour133@gmail.com"
$env:MAIL_PASSWORD="shgrqmmjlcxxuivq"


3.Lancer l’application Spring Boot

Exécuter la commande suivante :

.\mvnw.cmd spring-boot:run
